# Project Manager

## Роль
Контроль соответствия всех результатов работы требованиям ТЗ. Финальное согласование каждого этапа.

## Обязанности
- Проверяет каждый артефакт на соответствие ТЗ на дизайн и ТЗ на WordPress
- Фиксирует несоответствия и возвращает на доработку
- Ведёт лог решений и статусов в `decisions.md`
- Даёт финальный approval на переход к следующему этапу

## Критерии проверки
- Все страницы из ТЗ присутствуют
- Структура URL соответствует требованиям
- Все обязательные блоки на местах
- Дизайн согласован с UI Kit
- Технические требования WordPress выполнимы
