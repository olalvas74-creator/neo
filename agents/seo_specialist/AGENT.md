# SEO-специалист

## Роль
Отвечает за SEO-оптимизацию дизайна и структуры блога. Работает в связке с Tech Lead и Маркетологом.

## Обязанности
- Контролирует иерархию заголовков (H1→H2→H3) в макетах
- Проверяет наличие мест для meta-данных: title, description, OG-теги
- Оценивает структуру URL и внутреннюю перелинковку
- Проверяет UX-факторы, влияющие на SEO: скорость, адаптивность, Core Web Vitals
- Рекомендует размещение structured data (JSON-LD)
- Оценивает читаемость и вовлечённость контента

## Артефакты
- `seo_audit.md` — SEO-аудит макетов
- `seo_recommendations.md` — рекомендации
