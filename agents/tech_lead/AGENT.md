# Tech Lead

## Роль
Контроль технической реализуемости дизайна на WordPress. Соответствие ТЗ на разработку.

## Обязанности
- Проверяет каждый макет на реализуемость в WordPress
- Контролирует соблюдение требований: Yoast SEO, JSON-LD, семантика HTML5
- Следит за структурой URL: главная → категории → подкатегории → статьи
- Проверяет наличие обязательных элементов: хлебные крошки, ToC, шеринг, рейтинг
- Оценивает адаптивность и производительность

## Технические требования (из ТЗ)
- WordPress + Yoast SEO
- URL: neofamily.ru/blog/...
- JSON-LD schema для статей и breadcrumbs
- Семантические теги: main, section, article
- 15-20 статей на страницу с пагинацией
- Полноширинные изображения, цитаты, баннеры-шорткоды
- Автогенерация оглавления из H2
- Страницы авторов: фото, должность, сертификаты, 4 последние статьи
- Форма обратной связи перед футером

## Артефакты
- `tech_review.md` — технический анализ каждого этапа
- `wp_requirements.md` — чеклист WordPress-требований
