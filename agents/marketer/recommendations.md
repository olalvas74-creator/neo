# Рекомендации для блога NeoFamily

**На основе анализа конкурента:** Умскул Журнал
**Дата:** 2026-03-17

---

## 1. Что взять у конкурента (работает хорошо)

### 1.1. Структура контента
- **Лонгриды 2000-3000 слов** -- это работает для SEO. Минимум 1500 слов на статью.
- **Оглавление в начале статьи** -- обязательно, улучшает поведенческие факторы и дает Featured Snippets.
- **Иерархия H2/H3** -- четкая структура, удобная для чтения и сканирования.
- **Блок "Читай по теме"** в конце -- 3-4 похожие статьи для снижения отказов.

### 1.2. Тон и стиль
- **Обращение на "ты"** -- наша ЦА это школьники, им близок неформальный тон.
- **Заголовки с числами и двоеточием** -- доказанно высокий CTR: "7 ошибок в сочинении ЕГЭ: как их избежать".
- **Микс экспертного и разговорного** -- учебные статьи экспертные, лайфстайл -- разговорные.

### 1.3. Категоризация
- Разделение на **ЕГЭ / ОГЭ / Лайфстайл / Родителям** -- проверенная модель.
- **Авторство** на каждой статье -- создает доверие, работает на E-E-A-T.

### 1.4. Конверсионные элементы
- **2 CTA-баннера внутри статьи** (после вступления + перед заключением) -- оптимальное количество.
- **Нативные упоминания** сервисов платформы в тексте.

## 2. Что сделать лучше конкурента

### 2.1. Навигация и поиск

**Проблема у конкурента:** нет фильтров по предметам, нет поиска.

**Наше решение:**
- Добавить **фильтры по предметам** (биология, химия, русский язык) внутри категорий ЕГЭ/ОГЭ
- Реализовать **поиск по блогу** с автодополнением
- Добавить **теговую систему** -- предмет, тип контента (теория, практика, лайфхак), сложность

### 2.2. Sidebar (боковая панель)

**Проблема у конкурента:** нет sidebar, упущены возможности.

**Наше решение (на страницах статей):**
- Sticky-оглавление (следует при скролле)
- Блок "Популярное по [предмету]"
- CTA-виджет записи на курс
- Прогресс-бар чтения вверху страницы

### 2.3. Вовлечение и комьюнити

**Проблема у конкурента:** нет комментариев, нет UGC.

**Наше решение:**
- **Комментарии** через ВК-виджет или собственную систему (модерируемые)
- **Рейтинг статьи** ("Была ли статья полезна?") -- простая оценка
- **Шаринг** -- кнопки ВК, Telegram, WhatsApp, копирование ссылки
- **Время чтения** на карточках ("5 мин чтения")

### 2.4. Retention-механики

**Проблема у конкурента:** нет email-рассылки, слабый возврат пользователей.

**Наше решение:**
- **Email-подписка** -- "Получай полезные материалы для подготовки к ЕГЭ" с лид-магнитом (чек-лист, шпаргалка)
- **Telegram-бот/канал** -- автопостинг новых статей
- **Push-уведомления** (через OneSignal или аналог) -- ненавязчивые, 2-3 раза в неделю
- **Серия писем** по предметам -- автоматическая цепочка после подписки

### 2.5. Интерактив и геймификация

**Что у конкурента:** только тесты-квизы.

**Наше решение (расширенное):**
- **Тесты-квизы** по предметам (как у конкурента, это работает)
- **Калькулятор баллов ЕГЭ** -- интерактивный инструмент ("Узнай, в какой вуз ты проходишь")
- **Чек-листы подготовки** -- скачиваемые PDF в обмен на email
- **Таймер до ЕГЭ/ОГЭ** -- виджет на главной и в sidebar

## 3. Контент-стратегия для NeoFamily

### 3.1. Категории блога

| Категория | Описание | Приоритет |
|-----------|----------|-----------|
| **ЕГЭ** | Подготовка по биологии, химии, русскому языку | Высший |
| **ОГЭ** | Подготовка по тем же предметам | Высокий |
| **Лайфхаки** | Тайм-менеджмент, мотивация, память, стресс | Средний |
| **Родителям** | Как помочь ребенку, выбор вуза, финансы | Средний |
| **Новости** | Расписание экзаменов, изменения в заданиях | Высокий (сезонно) |

### 3.2. Типы контента

1. **SEO-лонгриды** (2000-3000 слов) -- основа, таргет на поисковые запросы
2. **Практические гайды** -- "Как написать сочинение ЕГЭ: пошаговая инструкция"
3. **Списки и подборки** -- "10 частых ошибок в ЕГЭ по биологии"
4. **Тесты/квизы** -- "Проверь свои знания по химии"
5. **Шпаргалки** -- компактные справочные материалы (таблицы, формулы)
6. **Новостные** -- расписания, изменения, даты

### 3.3. Частота публикаций

- **Старт:** 3 статьи в неделю (1 по каждому предмету)
- **Рост:** 5 статей в неделю через 2 месяца
- **Приоритет:** SEO-лонгриды по высокочастотным запросам

### 3.4. Примеры заголовков для NeoFamily

**Биология:**
- "Генетика для ЕГЭ: 12 типов задач с разбором"
- "Митоз и мейоз: шпаргалка, которая реально помогает"
- "Как запомнить все отделы растений: метод, который работает"

**Химия:**
- "ОВР в ЕГЭ по химии: пошаговый алгоритм расстановки коэффициентов"
- "Таблица Менделеева: 7 лайфхаков для запоминания"
- "Органическая химия с нуля: гайд для тех, кто запутался"

**Русский язык:**
- "Сочинение ЕГЭ на 24 балла: структура + 5 примеров"
- "Пунктуация в ЕГЭ: все правила в одной статье"
- "Паронимы ЕГЭ-2026: полный список с примерами"

## 4. Дизайн и UX блога

### 4.1. Карточки статей (улучшенные vs конкурент)

```
+----------------------------------+
|  [Изображение 16:9]             |
|                                  |
|  [Категория]  [5 мин чтения]   |
|  Заголовок статьи в две строки  |
|  Краткое описание на 2 строки...|
|  [Аватар] Автор  |  15 мар 2026|
+----------------------------------+
```

**Отличия от конкурента:**
- Добавлено время чтения
- Соотношение изображения 16:9 (не квадрат) -- лучше для карточек
- Описание ограничено 2 строками (не обрезается посередине слова)

```

  [CTA-баннер после вступления]  - Популярное
```

### 4.3. Фирменный стиль в блоге

- **Основной цвет CTA:** розовый/красный NeoFamily (#E91E63 или близкий)
- **Фон:** светло-серый (#F5F5F5) для контента, белый для карточек
- **Шрифт заголовков:** Comfortaa -- округлый, дружелюбный, выделяет бренд
- **Шрифт текста:** Nunito -- хорошая читаемость длинных текстов
- **Метки категорий:** цветовое кодирование (биология -- зеленый, химия -- синий, русский -- красный)

## 5. Технические рекомендации (WordPress)

### 5.1. Обязательные плагины
- **Yoast SEO** или **Rank Math** -- SEO-оптимизация, schema-разметка
- **WP Rocket** или **LiteSpeed Cache** -- скорость загрузки
- **ShortPixel** -- оптимизация изображений (WebP)
- **Easy Table of Contents** -- автоматическое оглавление
- **Social Warfare** или **Flavor** -- кнопки шаринга

### 5.2. Schema-разметка (обязательно)
- `Article` / `BlogPosting` -- для каждой статьи
- `BreadcrumbList` -- хлебные крошки
- `Organization` -- данные NeoFamily
- `FAQPage` -- для статей с вопросами-ответами (Featured Snippets)

### 5.3. Core Web Vitals
- LCP < 2.5 сек (оптимизация изображений, CDN)
- CLS < 0.1 (зафиксированные размеры изображений)
- FID < 100 мс (минимум JS на первом экране)
