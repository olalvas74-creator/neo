# Анализ конкурента: Умскул Журнал

**URL:** https://umschool.net/journal/
**Дата анализа:** 2026-03-17

---

## 1. Общее описание

Умскул Журнал -- контентный хаб образовательной платформы Умскул. Журнал выполняет две функции: привлечение органического трафика (SEO) и конверсия читателей в учеников платформы. Контент ориентирован на школьников 8-11 классов, абитуриентов и частично родителей.

---

## 2. Структура главной страницы

| Блок | Описание |
|------|----------|
| Шапка | Логотип, кнопка "Войти", горизонтальное меню категорий |
| Hero-баннер | Промо текущей акции (летняя школа / флеш-финал), яркий, с CTA-кнопкой |
| "Актуальное" | 6 последних статей в сетке карточек |
| Блоки по категориям | ЕГЭ, ОГЭ, Гайд по жизни, Кем стать, Тесты, POV -- каждый со своими статьями |
| Промо-блок | Форма записи на курс (имя, телефон, класс) |
| Подвал | Контакты, соцсети (ВК, YouTube, Telegram), юридическая информация |

**Вывод:** страница длинная, насыщенная, построена по принципу "витрины контента" -- показать максимум тематик, зацепить разных пользователей.

---

## 3. Навигация и категории

### Основные разделы меню (9 шт.):

1. **ЕГЭ** -- подготовка к ЕГЭ по предметам
2. **ОГЭ** -- подготовка к ОГЭ
3. **POV** -- трендовый лайфстайл-контент для подростков
4. **Гайд по жизни** -- психология, здоровье, лайфхаки
5. **Кем стать** -- профориентация, карьера
6. **Тесты** -- интерактивные тесты/квизы
7. **Будь в курсе** -- новости образования
8. **Родителям** -- контент для родителей
9. **Средняя школа** -- для 5-8 классов

### Внутри категорий:
- Нет фильтров по предметам (слабое место)
- Пагинация: классическая (28 страниц в ЕГЭ)
- 12 статей на страницу

---

## 4. Карточки статей

Каждая карточка содержит:
- **Изображение** -- крупное, 1320x1002 px, яркое, привлекающее внимание
- **Категория** -- цветная метка (ЕГЭ, ОГЭ, Тесты и т.д.)
- **Заголовок** -- крупный, кликбейтный, часто с кавычками или двоеточием
- **Автор** -- имя + аватар
- **Дата публикации**
- **Краткое описание** -- 100-150 символов

**Чего нет:** счетчик просмотров, время чтения, рейтинг.

---

## 5. Структура отдельной статьи

| Элемент | Реализация |
|---------|-----------|
| Заголовок | H1, крупный |
| Автор | Имя + аватар + ссылка на профиль |
| Дата | Под заголовком |
| Оглавление | В начале статьи ("Содержание") |
| Текст | H2/H3 подзаголовки, списки, цитаты, примеры |
| Изображения | Внутри текста, иллюстрации |
| CTA-баннеры | 2 шт. внутри статьи (после вступления и перед заключением) |
| Похожие статьи | Блок "Читай по теме" -- 3 статьи с превью |
| Комментарии | Отсутствуют |
| Шаринг | Только Telegram-бот, нет кнопок соцсетей |
| Хлебные крошки | Есть |
| Боковая панель | Отсутствует |

**Объем статей:** ~2500-3000 слов (лонгриды, хорошо для SEO).

---

## 6. CTA-элементы и конверсионные механики

1. **Hero-баннер** на главной -- промо текущего курса/акции
2. **Inline-баннеры** внутри статей (2 шт.) -- реклама курсов Умскул
3. **Форма лидогенерации** -- имя, телефон, класс (9-11), согласие на обработку данных
4. **Кнопка "Войти"** в шапке
5. **Ссылки на репетиторов** -- нативные упоминания в тексте статей (Умскул Репетиторы)
6. **Ссылки на соцсети** -- ВК, YouTube, Telegram в подвале

**Чего нет:** email-подписка на рассылку, push-уведомления, pop-up при уходе.

---

## 7. SEO-оснащение

### Техническое:
- Schema.org разметка: BlogPosting, Organization, WebSite, BreadcrumbList
- Open Graph теги (заголовок, описание, изображение)
- Хлебные крошки на всех страницах
- Человекочитаемые URL: `/journal/oge/sochinenie-po-russkomu-yazyku-na-oge/`
- Мета-описания для каждой статьи

### Контентное:
- Длинные статьи (2500-3000 слов)
- Оглавление с якорными ссылками
- H2/H3 иерархия подзаголовков
- Ответы на конкретные запросы ("расписание ЕГЭ-2026", "баллы по заданиям ЕГЭ")

## 8. Стиль и тон контента

### Заголовки -- приемы:
- **Двоеточие-формула:** "Инфляция для ЕГЭ по обществознанию: за 5 минут"
- **Кавычки/цитаты:** "Дави на газ: как подростку получить водительские права"
- **Числа:** "24 игры для изучения английского языка", "10 платформ с бесплатными курсами"
- **Вопросы:** "Как записаться на ЕГЭ-2026"
- **Трендовые слова:** "имба", "зумеры", "бабушкинг"

### Тон:
- Разговорный, на "ты"
- Прямое обращение к читателю ("Знакомая ситуация?", "Казалось бы...")
- Молодежный сленг в лайфстайл-контенте
- Экспертный в учебных материалах
- Без академической сухости

---

## 9. Сильные стороны конкурента

1. **Широкий контентный охват** -- от ЕГЭ до лайфстайла, цепляет разные интересы ЦА
2. **Сильное SEO** -- структурированные данные, лонгриды, правильные URL
3. **Тон подачи** -- говорят на языке аудитории, не менторствуют
4. **Нативная конверсия** -- баннеры курсов органично вписаны в статьи
5. **Регулярность** -- свежий контент, привязка к актуальным датам (расписание экзаменов, ВПР)
6. **Интерактив** -- тесты/квизы повышают вовлеченность и время на сайте
7. **Авторство** -- каждая статья подписана, создает доверие

---

## 10. Слабые стороны конкурента

1. **Нет фильтров по предметам** внутри категорий ЕГЭ/ОГЭ -- неудобно искать нужный предмет
2. **Нет бокового меню/sidebar** -- упущена возможность для дополнительной навигации и CTA
3. **Нет комментариев** -- нулевой UGC, нет сообщества
4. **Нет email-рассылки** -- слабая retention-механика
5. **Нет времени чтения** на карточках -- мелочь, но влияет на CTR
6. **Нет поиска по сайту** (или он скрыт) -- затрудняет навигацию
7. **Слабый шаринг** -- только Telegram, нет кнопок ВК/WhatsApp
8. **Нет dark mode** -- для школьников, которые читают ночью, это минус
9. **Главная перегружена** -- слишком длинная, много блоков, может утомлять
