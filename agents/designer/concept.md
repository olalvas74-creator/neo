**Уровень 2 (Category bar):**

### Визуальный стиль — блок за блоком

**H1:**
- Comfortaa Bold 36px #1D1E1B
- Text-align: center
- Max-width: 700px, margin 0 auto
- Padding-top: 24px от header

**Hero-баннер:**
- Ширина: 100% контентной области (1200px)
- Высота: 400px
- Border-radius: 32px
- Фоновое изображение WebP, object-fit cover
- Overlay: линейный градиент снизу rgba(29,30,27,0.6) -> transparent на 60%
- Контент поверх overlay внизу слева: padding 40px
- Заголовок акции: Comfortaa Bold 32px #FFFFFF
- Подзаголовок: Nunito Regular 18px rgba(255,255,255,0.85)
- CTA-кнопка: фон #E9585A, цвет #FFFFFF, Nunito Bold 18px, padding 14px 32px, border-radius 16px, margin-top 16px
- Margin-top от H1: 24px

**Секции категорий:**
- Чередование фонов: #FFFFFF / #F9F8FB
- Padding секции: 48px 0
- Заголовок секции: Comfortaa SemiBold 26px #1D1E1B, слева
- Ссылка "Читать ещё": Nunito SemiBold 16px, цвет категории, справа на уровне заголовка. Стрелка -> анимируется вправо на 4px при hover
- Accent bar: полоса 4px шириной 40px цвета категории под заголовком секции, margin-top 8px
- Сетка карточек: 4 колонки, gap 24px, margin-top 24px от accent bar

**Промо-баннер / подписка:**
- Аналогичен CTA-баннеру, но full-width (1200px)
- Высота: ~200px
- Вариант подписки: заголовок "Подпишись на рассылку" + поле email + кнопка "Подписаться"

### Spacing (сводка)

| Между чем | Отступ |
|-----------|--------|
| Header -> H1 | 24px |
| H1 -> Hero | 24px |
| Hero -> первая секция | 48px |
| Между секциями категорий | 0px (padding внутри каждой 48px top/bottom) |
| Заголовок секции -> accent bar | 8px |
| Accent bar -> сетка карточек | 24px |
| Последняя секция -> промо-баннер | 48px |
| Промо-баннер -> блок обратной связи | 48px |
| Блок обратной связи -> footer | 0px |

### Особые решения

- **Компактность:** у конкурента главная длинная и монотонная. У нас чередование белого и серого фона для визуального ритма + 4 карточки вместо бесконечной ленты
- **Accent bar под заголовком секции** в цвете категории — мгновенная визуальная навигация
- **H1 присутствует** (у конкурента его нет на главной)
- **Hero — не карусель**, одно статичное изображение для LCP < 2.5s

### Ключевые отличия мобильной адаптации

**Навигация (mobile < 768px):**
- Category bar заменяется на **горизонтальный скролл категорий** (не бургер-меню). Школьники привыкли к горизонтальному свайпу (Stories, TikTok). Бургер = лишний клик = потеря.
- Горизонтальный скролл: overflow-x auto, scrollbar скрыт (-webkit-scrollbar: none), padding 8px 16px, gap 8px, snap: scroll-snap-type: x mandatory, каждый таб scroll-snap-align: start
- Бургер-меню используется только для навигации основного сайта (Банк заданий, Магазин, Войти)

**Шрифты:**
- Используются только 2 семейства: **Comfortaa** (заголовки) + **Nunito** (всё остальное: body, мета, подписи, кнопки)
- Inter полностью заменён на Nunito для снижения нагрузки (2 семейства вместо 3 = меньше woff2-файлов, быстрее LCP)

**Карточки (mobile < 768px):**
- Описание в карточке: 16px (не 14px) — для читаемости на маленьких экранах

**Оглавление (mobile/tablet < 1024px):**
- Inline ToC отображается перед первым H2 (на desktop скрыт, т.к. есть sidebar)

**Sticky шеринг:**
- Скрыт на экранах < 1200px
- На mobile шеринг доступен только внизу статьи
