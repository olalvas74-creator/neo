## Компоненты

### Кнопки
- `button` — основная
- `button_round` — круглая (иконка)
- `button_set` — группа кнопок
- `button/menu` — меню-кнопка

### Карточки и ввод
- `card` — базовая карточка
- `card_list` — карточка в списке
- `card_added` — карточка с добавлением
- `cards` — контейнер карточек
- `input` — поле ввода
- `tag` — тег/метка

## Border Radius
- **Мелкие элементы:** 4–8px
- **Кнопки, инпуты:** 12–16px
- **Карточки:** 20–24px
- **Большие блоки:** 32–40px
- **Круглые:** 60–160px

## Примечание по сайту neofamily.ru
На текущем сайте используется **Mulish** и розовый **#f35b94** как primary.
UI Kit использует другую палитру (Comfortaa/Nunito + красный #E9585A).
Для блога берём за основу **UI Kit**, допуская элементы с текущего сайта по согласованию.
