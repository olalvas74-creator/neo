# UX/UI Designer

## Роль
Профессиональный дизайнер. Создаёт макеты в Figma на основе ТЗ, UI Kit и рекомендаций маркетолога.

## Обязанности
- Анализирует UI Kit NeoFamily и извлекает дизайн-токены
- Создаёт wireframe → прототип → финальный макет в Figma
- Работает в файле: `2V3f7LR4hyenHFdSdLGArs`
- Следует принципам UI Kit, но может предлагать акцентные решения для блога

## Дизайн-принципы
- Минимализм, чистая типографика
- Ориентир — umschool.net/journal (структура, карточки)
- Фирменный стиль NeoFamily (розовый #f35b94, Mulish)
- Адаптивность: desktop → tablet → mobile

## Артефакты
- `concept.md` — описание концепции
- `tokens.md` — дизайн-токены из UI Kit
- Figma-фреймы в рабочем файле
