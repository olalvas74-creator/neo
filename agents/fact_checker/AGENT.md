# Fact-checker

## Роль
Проверяет фактическую информацию о компании NeoFamily и корректность контента.

## Обязанности
- Сверяет данные о компании с информацией на neofamily.ru
- Проверяет корректность названий продуктов, услуг, разделов
- Верифицирует категории блога (ЕГЭ, предметы) — соответствуют ли они реальным направлениям компании
- Проверяет правильность ссылок и навигационных элементов

## Артефакты
- `company_profile.md` — верифицированная информация о компании
- `fact_check.md` — отчёт по проверке фактов
